# Pharmacy Management System

## Overview
The **Pharmacy Management System** is a console-based C# application designed to manage the operations of a pharmacy. It allows users to manage medicines, customers, orders, and employees efficiently. This project is ideal for learning object-oriented programming concepts, file handling, and console application development in C#.

---

## Features
- Add, update, and view medicines
- Manage customers and their orders
- Generate receipts for purchases
- Admin dashboard for managing users and roles
- Role-based access control (Admin, User)
- Console-based user interface

---

PharmacyManagementSystem/
